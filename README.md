## 모델 설명
1. 목적 
2. 사용방식
   - 사진과 환경 정보를 입력
3. 모델 구조
   - 사진 모델
     + 사진을 업로드하면 분석하여 부위 이름 및 생육 크기를 출력
     + 만약 사진이 잎이라면 질병 여부와 함께 감염시 해당 질병을 출력
     + 만약 열매라면, 해당 열매가 익었는지 익지 않았는지 판별
   - 환경 모델
     +예상 수확 결과를 출력
